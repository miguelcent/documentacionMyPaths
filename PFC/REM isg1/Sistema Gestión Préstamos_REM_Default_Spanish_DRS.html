    4.1.1 Alta de Cliente
    SOP-0001 Alta de Cliente
    Tipo del resultado Set(Tuple(String,String,String,Integer))
    Versión 1.0 ( 06/06/2010 )
    Autores
    • Carmen María Sivianes Burgos
    Dependencias
  • [UC-0001] Alta de cliente
    • Descripción Da de alta un nuevo cliente, necesario para pedir un préstamo. Hacen falta los datos comunes de una persona, tales como nombre,dni, dirección y telefono.
    Parámetros idCliente : Cliente -- el cliente es dado de alta en el banco para poder pedir un préstamo.
    Expresiones de precondición No existe el cliente: el cliente no existe en la base de datos.
    Expresiones de precondición (OCL) No existe el cliente: Cliente.allInstances - > exclude(idCliente)
    Expresiones de postcondición Cliente dado de alta: el cliente ya esta dado de alta en la base de datos.
    Expresiones de postcondición (OCL) Cliente dado de alta: let cliente: Cliente.allInstances() -> any (c|c.idCliente = idCliente)
    Comentarios Ninguno

    4.1.2 Cancelar Préstamo

    SOP-0002 Cancelar Préstamo
    Tipo del resultado Boolean
    Versión 1.0 ( 06/06/2010 )
    Autores
    • Carmen María Sivianes Burgos
    Dependencias
  • [UC-0002] Cancelar préstamo
    • Descripción Despues de que el sistema nos proporciones las tasas y el total a pagar por el cliente, el asesor formaliza la cancelacion del préstamo guardando la fecha, poniendo el préstamo como cancelado e informando al cliente de que la operación ha ido bien.
    Parámetros idCliente : Cliente -- identificador único de cliente.
    Expresiones de precondición Existencia de Préstamo: el préstamo existe y queda dinero por pagar.
    Expresiones de precondición (OCL) Existencia de Préstamo: Cliente.prestamo.allInstances()-> one (c| c.númeroPréstamo=númeroPréstamo)
    Expresiones de postcondición Cancelación de Préstamo: el préstamo es cancelado y se procede a formalizarlo.
    Expresiones de postcondición (OCL) Cancelación de Préstamo: let prestamo: Prestamo.allInstances() -> any (p|p.númeroPréstamo = númeroPréstamo) in prestamo.formalizado = True
    Comentarios Ninguno

    4.1.3 Registrar Aceptación Préstamo

    SOP-0003 RegistrarAceptación
    Versión 1.0 ( 08/06/2010 )
    Autores
    • María De Miguel Jiménez
    Dependencias
  • [UC-0003] Registrar aceptación de un préstamo
    • Descripción Se autoriza un préstamo.
    Parámetros númeroPréstamo : String -- el identificador único de un préstamo.
    Expresiones de precondición Existencia de Préstamo: el préstamo existe y no está autorizado.
    Expresiones de precondición (OCL) Existencia de Préstamo: Prestamo.allInstances()-> one (p| p.númeroPréstamo=númeroPréstamo and p.autorizado=false)
    Expresiones de postcondición Autorización de préstamo: el préstamo es autorizado.
    Expresiones de postcondición (OCL) Autorización de préstamo: let prestamo: Prestamo.allInstances() -> any (p|p.númeroPréstamo = númeroPréstamo) in prestamo.autorizado = True
    Comentarios Ninguno

    SOP-0004 ConsultarPréstamo
    Tipo del resultado Tuple (Integer, Integer, Date)
    Versión 1.0 ( 08/06/2010 )
    Autores
    • María De Miguel Jiménez
    Dependencias
  • [UC-0003] Registrar aceptación de un préstamo
    • Descripción Se obtienen los datos de un préstamo.
    Parámetros númeroPréstamo : String -- el identificador único de un préstamo.
    Expresiones de precondición Existencia del préstamo: el préstamo existe.
    Expresiones de precondición (OCL) Existencia del préstamo: Préstamo.allInstances() -> one (p| p.númeroPréstamo = númeroPréstamo)
    Expresiones de postcondición Devuelve el préstamo: se devuelven los datos del préstamos.
    Expresiones de postcondición (OCL) Devuelve el préstamo: let prestamo: Prestamo.allInstances() -> any (p|p.númeroPréstamo= númeroPréstamo) in result = Tuple(idCliente:Integer=idC, idAvalista:Integer=idA,fechaSolicitud:Date=fechaS)
    Comentarios Se supone que existe un atributo identificador por defecto de los tipos Cliente y Avalista denominado id.

    4.1.4 Alta de Préstamo

    SOP-0005 AltaPréstamo
    Tipo del resultado String
    Versión 1.0 ( 09/06/2010 )
    Autores
    • María De Miguel Jiménez
    • Miguel Centeno Muñoz
    Dependencias
  • [UC-0007] Alta de Préstamo
    • Descripción Se da de alta un nuevo préstamo.
    Parámetros idCliente : Real -- el identificador único de un cliente.
     importe : Real -- el importe por el que se solicita el préstamo.
    Expresiones de precondición Existencia del cliente: el cliente existe.
     Existencia del Asesor: el asesor existe.
    Expresiones de precondición (OCL) Existencia del cliente: Cliente.allInstances()->one(c|c.idCliente=idCliente)
    Existencia del Asesor: Asesor Financiero.allInstances()->one(a|a.idAsesor=idAsesor)
    Expresiones de postcondición Creación del préstamo: se crea un nuevo préstamo.
    Expresiones de postcondición (OCL) Creación del préstamo: let cliente:Cliente.allInstances()->any(c|c.idCliente=idCliente) asesor:Asesor.allInstances()-<any(a|a.idAsesor=idAsesor) prestamo:Prestamo.allInstances()->any(p|p.númeroPréstamo=númeroPréstamo) in prestamo.oclIsNew() = True and prestamo.importe=importe and prestamo.fechaSolicitud=Date.Today() and prestamo.Cliente=cliente and prestamo.AsesorFinanciero= asesor and asesor.prestamo->includes(prestamo) and cliente.prestamo->includes(prestamo) and result=númeroPréstamo
    Comentarios Se supone que existe un atributo identificador por defecto del tipo Cliente denominado id.
